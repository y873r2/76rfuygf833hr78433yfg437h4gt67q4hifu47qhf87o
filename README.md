<section>

  <h2>1. Om arten</h2>
  <div class="box">
    <p><b>Navn:</b> Kveke (Elymus repens)</p>

    <p><b>Hvordan ser den ut?</b><br>
    Kveke er et gress med lange grønne blader og små aks på toppen. Den sprer seg raskt under bakken med røtter (jordstengler).</p>

    <p><b>Spesielle egenskaper:</b><br>
    Kveke er veldig hardfør og sprer seg raskt. Den kan overleve selv om den blir klippet ned.</p>

    <p><b>Status i naturen:</b><br>
    Kveke er vanlig i Norge og regnes ofte som et ugress i åkre og hager.</p>
  </div>

  <h2>2. Økosystem og samspill</h2>
  <div class="box">
    <p><b>Økosystem:</b><br>
    Kveke finnes ofte i jordbruksområder, enger og veikanter.</p>

    <p><b>Biotiske faktorer (levende):</b><br>
    Andre planter, insekter, meitemark og dyr som beiter.</p>

    <p><b>Abiotiske faktorer (ikke-levende):</b><br>
    Sollys, vann, jordtype, temperatur og næringsstoffer i jorda.</p>

    <p><b>Samspill:</b><br>
    Kveke konkurrerer med andre planter om lys og næring. Den påvirkes av klima og jord, og påvirker økosystemet ved å dekke store områder raskt.</p>
  </div>

  <h2>3. Næringskjede</h2>
  <div class="box">
    <p><b>Kveke sin rolle:</b> Produsent</p>

    <p><b>Hvorfor?</b><br>
    Kveke lager sin egen mat gjennom fotosyntese.</p>

    <p><b>Næringskjede eksempel:</b><br>
    Kveke → Gresshopper → Frosk → Fugl</p>

    <p><b>Andre begreper:</b><br>
    - Produsent: Kveke<br>
    - Konsument: Insekter, dyr<br>
    - Nedbryter: Sopp og bakterier</p>
  </div>

  <h2>4. Fagbegreper</h2>
  <div class="box">
    <p>
      Fotosyntese, celleånding, økosystem, biotiske faktorer, abiotiske faktorer, produsent, konsument og nedbryter brukes for å forklare hvordan naturen henger sammen.
    </p>
